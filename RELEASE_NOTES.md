# Audiswift v1.0.0 Release Notes

Welcome to the first official release of **Audiswift**, a premium, high-fidelity macOS music player designed for the Audius ecosystem.

## Features

### 🎵 Core Playback
- **Responsive Controls**: Immediate play/pause and track skipping.
- **Persistent Player Bar**: Real-time track information, artwork, and duration tracking.
- **High-Fidelity Audio**: Optimized playback engine for a smooth listening experience.

### 🌈 Dynamic Theme System
- **5 Curated Themes**: Switch between Default, Classic, Matrix, Sunset, and Ocean styles.
- **Instant Application**: UI and visualizers update immediately to match your style.
- **Neon Aesthetics**: Includes the fan-favorite Matrix theme with neon-green visuals.

### 📊 Professional Audio Visualizers
- **Real-time Frequency Response**: Smooth, 60fps vertical bars.
- **Oscilloscope Waveform**: Beautifully animated waveform traces for a vintage aesthetic.
- **Seamless Transitions**: Instantly switch between visualization modes.

### 🔍 Discovery & Integration
- **Audius API Integration**: Browse trending tracks across multiple genres.
- **Smart Filtering**: Filter by Time (Week, Month, All Time) and Genre.
- **Genre Cards**: Quickly discover new music through a colorful genre-based interface.

## Performance & Stability
- **Built for Apple Silicon**: Native performance and low resource usage.
- **Production Ready**: Zero known crashes and smooth navigation across all screens.
